Sushi Lovers

Sushi Lovers es una revista que nace de un mini-proyecto juntando varias herramientas. En un comienzo nos introducen un branding de una marca ficticia creando un logotipo de un restaurante asiático de nombre libre. En mi caso escogí "Kohaku Bar Sushi" y posteriormente diseñé Sushi Lovers, una magazine fictial mensual para mezclar elementos e introducir el logo ya diseñado.

En este proyecto existe la creación de logos, edición de imagen, mockups y maquetación de una revista.

Ámbito: Branding, Diseño Digital, Maquetación.

Herramientas: Illustrator, Photoshop, InDesign.

Planificación

Tras leer el branding y definir un logotipo. Se buscan referencias de otras revistas de gastronomía, con recomendaciones de sitios, entrevistas y buscando un lenguaje parecido

Estructura de la Revista

¿Cuántas páginas habrá?, ¿Qué secciones?

Con una tabla de contenidos y manteniendo el diseño entre las mismas páginas, se busca una unión y sentido entre ellas.

Diseño de Plantilla

Creando páginas maestras se sigue la estructura anteriormente elaborada, se generan también estilos de texto para títulos, textos y pie de páginas.

Composición Visual

Mediante el uso de imágenes y gráficos (Como el logotipo) se mezclan junto al texto usando tipografías legibles y que complementen el tema de la revista. Siempre considerando la jerarquía tipográfica.

Integración y Revisión

Importación de textos e imágenes y colocadas de forma en la que el texto fluya y las imágenes estén bien alineadas. Finalmente se hacen revisiones y correciones. Si fuese una revista para imprimir, abría que realizar pruebas de papel, color, sangrado...
